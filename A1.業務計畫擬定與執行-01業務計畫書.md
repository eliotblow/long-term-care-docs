A
經營管理效能


A.1
人力資源管理
1. 員工招募與任用： 依據長照人員規範，辦理招募、面試與聘用作業。
2. 教育訓練： 規劃年度教育訓練計畫，新進人員完成職前訓練；在職人員每年完成至少20小時繼續教育。
3. 績效管理： 每半年進行一次人員績效考核，並依結果發放獎勵金。
4. 員工關懷與晉升： 建立暢通申訴管道，並規劃人員晉升與發展路徑。
建立穩定、專業且具向心力的服務團隊，提升整體照顧服務品質。
A.2
財務與財產管理
1. 經費控管： 每月完成經費收支分析，確保財務穩健。
2. 財產管理： 依據財產管理辦法，執行年度財產盤點、登錄與報廢作業。
確實控管機構財務與財產狀況，確保資源有效運用及永續經營。
A.3
行政與品質管理
1. 工作手冊修訂： 年度開始時檢視並修訂工作手冊，以符合實際作業需求。
2. 品質監測會議： 每月定期召開服務品質會議，檢討服務流程、意外事件、家屬意見等。
3. 主管機關查核改善： 針對主管機關之查核、輔導、評鑑缺失，列管追蹤並確實改善。
建立標準化作業流程，透過持續性的品質監測與改善，提升服務滿意度。
A.4
社區連結與行銷
1. 社區資源盤點： 每半年更新一次社區資源網絡圖。
2. 社區活動辦理： 依年度計畫辦理社區宣導、共融活動，並積極參與在地組織活動。
3. 服務資訊公開： 維護並更新機構網站/社群平台，公開透明服務資訊。
提升機構在社區的能見度與正面形象，開拓潛在個案來源。
B
專業照護品質


B.1
個案管理與照顧
1. 個案評估與計畫： 新案依流程進行跨專業評估；每6個月或視情況修正個案照顧計畫。
2. 跨專業會議： 每季至少辦理1次跨專業個案研討會，並追蹤決議事項。
3. 適應輔導： 針對新進個案或適應困難者，提供個別化輔導與支持。
提供即時、適切且個別化的全人服務，穩定服務對象身心狀況。
B.2
健康管理與促進
1. 健康監測與管理： 落實服務對象健康檢查、每日生理徵象量測及異常追蹤。
2. 感染管制作業： 依據感控手冊執行防疫措施，並定期召開會議檢討。
3. 意外事件預防與處理： 針對跌倒、噎嗆等高風險事件進行預防與管理，事後檢討分析並提出改善方案。
4. 營養膳食服務： 依服務對象個別需求提供適當餐點，並確保餐點具變化性。
延緩長者失能、促進健康，並有效預防與控制感染及意外事件。
B.3
活動與支持服務
1. 多元活動辦理： 每季規劃多元適性的動、靜態團體課程，並留存活動紀錄。
2. 自我照顧能力維持： 透過活動設計與日常協助，鼓勵並維持服務對象自我照顧能力。
3. 照顧者支持： 每季辦理家屬座談會或聯誼活動，並建立暢通的日常溝通管道。
豐富服務對象社交生活，提升生活品質，並減輕家庭照顧者壓力。
B.4
人員專業知能
1. 繼續教育規劃與執行： 依長照人員需求與法規，規劃年度繼續教育課程，確保每人完成20小時訓練。
2. 特殊訓練： 確保照服員完成失智症及身心障礙服務相關訓練。
3. 急救能力： 確保所有長照人員具備有效之急救證明文件。
提升人員全面性專業知能與工作成就感，確保服務品質與安全。
C
安全環境設備


C.1
公共安全維護
1. 消防安全： 每半年進行消防安全設備檢修申報，並每半年辦理一次消防演練。
2. 公安申報： 依規定時程辦理建築物公共安全檢查簽證申報。
3. 災害應變： 訂定緊急災害應變計畫，並定期演練。
4. 設施設備維護： 每月定期檢查維護器材、急救箱及緊急呼叫系統。
維護服務對象與工作人員的生命財產安全，確保災害發生時能有效應變。
C.2
環境衛生與設施
1. 環境清潔： 落實每日環境清潔、每季環境消毒。
2. 餐飲衛生： 確保廚房環境與餐具清潔，並落實食物檢體留存。
3. 飲用水安全： 每半年清洗水塔、每月保養飲水機、每季進行水質檢驗。
提供一個安全、舒適、衛生的服務環境，預防交叉感染。
D
個案權益保障


D.1
權益保障措施
1. 服務契約： 新進個案確實簽訂服務契約，每年與舊個案完成續約。
2. 收費與收據： 依主管機關核定標準收費，並開立詳實收據。
3. 申訴與意見反映： 公開申訴管道與流程，並依規定時效處理、回覆及分析檢討。
4. 滿意度調查： 每年辦理至少1次服務對象及家屬滿意度調查，並依結果提出改善方案。
5. 保險與個資： 依法投保公共意外責任險，並妥善保管個案資料。
尊重並保障服務對象的權利與隱私，建立互信的服務關係。
